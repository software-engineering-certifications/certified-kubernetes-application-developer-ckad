What is

What is Certified Kubernetes Application Developer (CKAD)?

The Cloud Native Computing Foundation (CNCF) and The Linux Foundation collaborated on the Certified Kubernetes Application Developer (CKAD) programme to assist expand the Kubernetes ecosystem through standardized training and certification. Users who pass the Certified Kubernetes Application Developer (CKAD) certification can design, construct, configure, and expose Kubernetes cloud native apps. To create, manage, and troubleshoot scalable apps and tools in Kubernetes, a CKAD can describe application resources and employ basic primitives.
